Base de datos SQL

En esta asignatura he aprendido a realizar un modelo del dominio ,así como sacar las tablas del modelo entidad-realción.Además en esta asignatura se aprende a realizar consultas a una base de datos mediante las clausulas SELECT, WHERE , FROM , GROUP BY entre otras. Después de saber manejar los principales sentencias LMD , se realizan subconsultas más complejas para realizar consultas más elaboradas.A su vez , se estudia PL-SQL en el cual se utiliza para realizar tareas que no se pueden realizar solo con SQL.Se estudian PROCEDURES, TRIGGERS y elementos que se pueden utilizar dentro de estos procedimientos como pueden ser cursores , para almacenar contenidos de las tablas o registros para posteriormente consultar la informacion de las tablas. Se introduce también en concepto de transacciones , asi como los distintos tipos de estados de la transacción despues de hacer sentencias LMD y LDD.
